A1 Beginner Course

Start your journey to learning Moroccan Darija with these foundational modules.

Course Modules

Module 0: Darija Sounds & Transcription

A1.0

Master the unique sounds of Darija and learn how to read the transcription system used in this course.

Start Module 0

Module 1: Greetings, Politeness & Introductions

A1.1

Learn basic greetings, introductions, essential polite phrases, and subject pronouns.

Start Module 1

Module 2: Verb 'To Be', Nouns & Gender

A1.2

Understand the 'zero copula' for is/am/are, learn about noun gender, and the definite article 'l-'.

Start Module 2

Module 3: 'To Have' & Possession

A1.3

Learn to say "I have" using `3ndi` and express ownership with possessive suffixes and `dyal`.

Start Module 3

Module 4: Numbers (1-100) & Telling Age

A1.4

Master counting from 1 to 100 and learn how to ask and state your age.

Start Module 4

Module 5: Demonstratives & Common Objects

A1.5

Learn to use "this," "that," "these," and "those," and identify common objects.

Start Module 5

Module 6: Adjectives & Agreement, Colors

A1.6

Learn how to describe nouns using adjectives, ensure agreement, and basic color vocabulary.

Start Module 6

Module 7: Verb 'To Want', Food & Drink

A1.7

Master the verb "to want" (`bghit`) and learn vocabulary for common food and drinks.

Start Module 7

Module 8: Present Tense (`ka-` verbs)

A1.8

Learn to talk about habits and regular actions using the `ka-` present tense marker.

Start Module 8

Module 9: Asking Basic Questions

A1.9

Learn to ask Yes/No questions with `Wash` and use basic question words (Who, What, Where, When, How Much?).

Start Module 9

Module 10: Days, Months & Time

A1.10

Learn the days of the week, months of the year, and how to tell time in Darija.

Start Module 10

Module 11: Family Members

A1.11

Learn vocabulary for immediate family members and how to talk about your family.

Start Module 11

Module 12: Prepositions of Place & Conjunctions

A1.12

Learn essential prepositions to describe location (in, on, under, next to...) and basic conjunctions (and, or).

Start Module 12

Module 13: Negation

A1.13

Master negation using `ma-...-sh` for verbs, `mashi` for nouns/adjectives, and `ma 3ndish`.

Start Module 13

Module 14: Imperatives (Commands)

A1.14

Learn how to give commands, both telling someone to do something and not to do something.

Start Module 14

Module 15: Shopping & Bargaining

A1.15

Apply your A1 skills in practical shopping scenarios, including essential bargaining phrases.

Start Module 15

Module 16: Locations & Directions

A1.16

Learn vocabulary for places and how to ask for and give basic directions.

Start Module 16

Module 17: Expressing Likes/Dislikes

A1.17

Learn the common structure `kay3jbni...` to talk about what you like and dislike.

Start Module 17

Module 18: A1 Review & Consolidation

A1.18

Review all the A1 topics, practice with mixed exercises, and prepare for the next level!

Start Module 18
