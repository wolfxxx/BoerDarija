B1 Intermediate Course

Deepen your Darija skills to navigate more complex conversations, express nuanced ideas, and understand a wider range of topics.

B1 Course Modules

Module 1: A2 Review & Advanced Past Tense

B1.1

Consolidate A2 knowledge and delve deeper into past tense, focusing on irregular, 'hollow', and 'weak' verbs, and narrative techniques.

Start Module 1

Module 2: The Imperfect (Habitual Past)

B1.2

Learn to describe habitual or ongoing actions in the past using the "kan + ka-verb" structure (e.g., "I used to...", "I was -ing").

Start Module 2

Module 3: Active & Passive Participles

B1.3

Understand and form active (doer) and passive (done-to) participles, and their use as adjectives or in simple passive voice.

Start Module 3

Module 4: Expressing Ability, Possibility & Permission

B1.4

Learn to use `qder` (can), `yemken` (maybe/it's possible), and phrases for asking/giving permission.

Start Module 4

Module 5: Expressing Obligation & Necessity

B1.5

Master `khass` (must/need), `Daruri` (necessary), `lazem` (it is necessary) to talk about obligations and give advice.

Start Module 5

Module 6: More Complex Conditionals

B1.6

Explore present and past unreal conditionals ("If I had..., I would...") and using `lu kan`.

Start Module 6

Module 7: Advanced Relative Clauses & `ashmn`

B1.7

Deepen your use of `lli` including with prepositions, and learn to use `ashmn` (which).

Start Module 7

Module 8: Derived Verb Forms I (Causative)

B1.8

Introduction to verb patterns (awzan), focusing on Form II (`fe33el`) for causative and intensive meanings.

Start Module 8

Module 9: Derived Verb Forms II (Reflexive/Passive)

B1.9

Explore Form V (`tfe33el`) and Form VII (`nfe3el`) for reflexive and passive meanings.

Start Module 9

Module 10: More Connectors & Discourse Markers

B1.10

Learn advanced connectors for contrast, cause/reason, result, and adding information.

Start Module 10

Module 11: Reported Speech (Simple)

B1.11

Learn to report what others said (`qal belli...`) and asked (`suwwlat wash...`).

Start Module 11

Module 12: Describing in Detail

B1.12

Practice using a wide range of adjectives, relative clauses, and comparisons for rich descriptions.

Start Module 12

Module 13: Expressing Emotions & Feelings

B1.13

Expand vocabulary for emotions and learn structures to express how you feel more precisely.

Start Module 13

Module 14: Discussing Social & Cultural Topics

B1.14

Gain vocabulary for traditions, holidays, and daily life, and practice expressing opinions on these topics.

Start Module 14

Module 15: Handling Problems & Complaints

B1.15

Learn phrases for expressing dissatisfaction and asking for help with simple problems.

Start Module 15

Module 16: Understanding Different Registers

B1.16

Develop an awareness of variations in Darija (formal, informal, influence of French/MSA).

Start Module 16

Module 17: Extended Narratives & Storytelling

B1.17

Practice telling longer, more detailed stories in the past, using varied vocabulary and connectors.

Start Module 17

Module 18: B1 Consolidation & Moving to B2

B1.18

Comprehensive review of B1 grammar and vocabulary, and guidance for B2 Upper Intermediate.

Start Module 18
