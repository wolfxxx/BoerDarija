A2 Elementary Course

Continue your Darija journey! This A2 course will help you build more complex sentences, expand your vocabulary, and engage in richer conversations.

A2 Course Modules

Module 1: A1 Review & Intro to Past Tense

A2.1

Refresh A1 basics and learn the past tense for regular verbs, including past time expressions.

Start Module 1

Module 2: Past Tense (Irregular Verbs)

A2.2

Master the past tense of common irregular verbs like "to be," "to have," "to go," "to see," and "to do."

Start Module 2

Module 3: Object Pronoun Suffixes

A2.3

Learn to use suffixes for "me," "you," "him/her/it," "us," "them" attached to verbs.

Start Module 3

Module 4: Past Events & Simple Narratives

A2.4

Combine past tenses and sequencing words to tell simple stories and recount past events.

Start Module 4

Module 5: Future Tense with `Ghadi`

A2.5

Learn to talk about future plans and intentions using `ghadi` and future time expressions.

Start Module 5

Module 6: More Prepositions

A2.6

Expand your knowledge of prepositions, focusing on time, movement, and purpose.

Start Module 6

Module 7: Advanced Adjectives & Descriptions

A2.7

Learn more adjectives for personality and appearance, use multiple adjectives, and intensifiers.

Start Module 7

Module 8: Comparative & Superlative Adjectives

A2.8

Learn how to make comparisons (more than, as...as) and express superlatives (the most).

Start Module 8

Module 9: Health & Body

A2.9

Learn vocabulary for body parts, common health issues, and doctor/pharmacy interactions.

Start Module 9

Module 10: Daily Routines & Hobbies (Extended)

A2.10

Describe daily activities in more detail, discuss hobbies, and use adverbs of frequency.

Start Module 10

Module 11: Transportation & Travel (Extended)

A2.11

Learn vocabulary for transport types, buying tickets, and describing simple journeys.

Start Module 11

Module 12: Larger Numbers & Quantities

A2.12

Learn numbers beyond 100 (hundreds, thousands), fractions, and more complex quantity expressions.

Start Module 12

Module 13: Relative Clauses with `lli`

A2.13

Learn to connect ideas and add more information about nouns using `lli` (that/which/who).

Start Module 13

Module 14: Simple Conditional "If"

A2.14

Learn to express conditions and possible outcomes using `ila` (if) for real future possibilities.

Start Module 14

Module 15: Opinions & Suggestions

A2.15

Learn how to express your opinions and make suggestions for activities or plans.

Start Module 15

Module 16: Review of Verb Tenses

A2.16

Consolidate your understanding of present, past, and future tenses and practice using them.

Start Module 16

Module 17: Cultural Insights & Common Expressions

A2.17

Learn common idiomatic expressions, simple proverbs, and more about cultural etiquette.

Start Module 17

Module 18: A2 Consolidation & Moving Forward

A2.18

Review all A2 topics, practice with mixed exercises, and prepare for the B1 Intermediate level!

Start Module 18
