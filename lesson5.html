Module 5: Demonstratives & Common Objects

Learn how to say "this," "that," "these," and "those," and identify common objects around you.

Demonstratives: Pointing Things Out

Demonstratives are words we use to point to specific people or things. In Darija, they change based on gender (masculine/feminine) and number (singular/plural), and also sometimes for proximity (near/far).

Near Demonstratives (This / These - for things close by)

English Darija Script Transcription Usage Audio
This (masculine) هادا hada For masculine singular nouns
This (feminine) هادي hadi For feminine singular nouns
These (plural) هادو hadu For plural nouns (both genders)
The "had-" prefix: You might also see `had` (هاد) used as a prefix before a noun to mean "this/these," e.g., `had l-ktab` (this book), `had t-tomobil` (this car), `had n-nas` (these people). This is very common and often interchangeable with `hada/hadi/hadu` + noun.

Far Demonstratives (That / Those - for things further away)

English Darija Script Transcription Usage Audio
That (masculine) هاداك hadak For masculine singular nouns
That (feminine) هاديك hadik For feminine singular nouns
Those (plural) هادوك haduk For plural nouns (both genders)
The "k" sound at the end of `hadak`, `hadik`, `haduk` often indicates distance.

Using Demonstratives with Objects

Now let's combine them! Remember the zero copula from Module 2 (Subject + Noun/Adjective).

English Darija Script Transcription Audio
This is a book. هادا كتاب. Hada ktab.
This is a table. هادي طبلة. Hadi tbla.
That is a car. هاديك طموبيل. Hadik ttomobil.
That is a chair. هاداك كرسي. Hadak kursi.
These are keys. (keys: سواروت swaret - plural) هادو سوارت. Hadu swaret.
This house is big. (Using had + noun) هاد الدار كبيرة. Had d-dar kbira.
Gender Agreement is Crucial:
  • `Hada` (this/masc) with `ktab` (book/masc).
  • `Hadi` (this/fem) with `tbla` (table/fem).
  • `Hadik` (that/fem) with `ttomobil` (car/fem).
  • When using `had` + noun + adjective, the adjective must agree with the noun's gender: `Had d-dar` (F) `kbira` (F).
