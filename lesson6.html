Module 6: Adjectives & Agreement, Colors

Learn how to describe things using adjectives, making them agree with nouns, and common color words.

What are Adjectives?

Adjectives are words that describe nouns (people, places, things, or ideas). For example, "big," "small," "happy," "red."

Key Rule in Darija: Adjectives usually come AFTER the noun they describe, and they must AGREE in gender (masculine/feminine) and number (singular/plural) with that noun.

Gender Agreement of Adjectives

You learned about noun gender in Module 2. Now let's see how adjectives change.

Many masculine adjectives can be made feminine by adding an "-a" sound at the end.

English Adjective Masculine Darija (Script) Masculine (Transcription) Feminine Darija (Script) Feminine (Transcription)
Big كبير kbir كبيرة kbira
Small صغير sghir صغيرة sghira
New جديد jdid جديدة jdida
Old (for things) قديم qdim قديمة qdima
Happy فرحان fr7an فرحانة fr7ana
Good / Beautiful زوين / مزيان zwin / mzyan زوينة / مزيانة zwina / mzyana

Examples with Nouns:

English Darija Phrase (Script) Darija Phrase (Transcription)
A big book (Masc. Noun + Masc. Adj.) كتاب كبير Ktab kbir .
A big car (Fem. Noun + Fem. Adj.) طموبيل كبيرة Ttomobil kbira .
A new house (Fem. Noun + Fem. Adj.) دار جديدة Dar jdida .
A happy boy (Masc. Noun + Masc. Adj.) ولد فرحان Weld fr7an .

Number Agreement of Adjectives (Plural)

When describing plural nouns, adjectives also need to become plural. A common way to make adjectives plural is by adding "-in" to the masculine singular form.

English Singular Masculine Adjective Plural Adjective (Darija Script) Plural Adjective (Transcription)
Big (plural) kbir ( كبير ) كبار kbar
Small (plural) sghir ( صغير ) صغار sghar
New (plural) jdid ( جديد ) جداد jdad
Happy (plural) fr7an ( فرحان ) فرحانين fr7anin
Good / Beautiful (plural) zwin ( زوين ) زوينين zwinin
Irregular Plurals: As you can see with "kbar," "sghar," and "jdad," some very common adjectives have irregular plural forms that don't just add "-in." These need to be memorized. However, the "-in" ending is a useful pattern for many others.
Agreement with "Broken Plurals": If a noun has a "broken plural" (irregular plural form, e.g., `ktab` (book) -> `ktub` (books)), the adjective describing it often takes the feminine singular form .
Example: كتوب كبار (ktub kbar - big books) or كتوب زوينة (ktub zwina - nice books - using fem. sing. adj.). This is a more advanced point, but good to be aware of. For A1, focusing on the `-in` plural for adjectives and simple regular noun plurals is a good start.

Examples with Plural Nouns:

(Plural nouns will be covered more formally later. For now, focus on the adjective form.)

  • Happy boys: ولاد فرحانين ( wlad fr7anin )
  • New cars: طموبيلات جداد ( ttomobilat jdad ) (using irregular plural adj.)

Colors (as Adjectives)

Colors are adjectives too, and they must agree in gender with the noun they describe. Some colors have distinct masculine and feminine forms.

Color Masculine (Script / Trans.) Feminine (Script / Trans.)
Black كحل / k7el كحلة / k7la
White بيض / byed بيضة / bida
Red حمر / 7mer حمرا / 7mra
Green خضر / khder خضرا / khdra
Blue زرق / zreq زرقا / zerqa
Yellow صفر / sfer صفرا / sfra
Orange ليموني / limuni (often invariable)
Pink وردي / غوز / wardi / ghoz (often invariable)

Examples with Colors:

  • A red car (F): طموبيل حمرا ( ttomobil 7mra )
  • A blue book (M): كتاب زرق ( ktab zreq )
